Detailed Description

Display driver for the nokia based pcd8544 displays.

This module implements a driver for the standard pcd8544 nokia display. This driver uses the pca bus abstraction layer and works on top of any provided bus_t compatible SPI abstraction layer. Have a look on the example for details of using the driver
