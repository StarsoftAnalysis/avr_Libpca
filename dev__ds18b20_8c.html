This function updates the persistent sensor configuration.

It updates the contents of the RAM scratchpad and copies them explicitly to the internal ROM. By default (force = 0) it won't rewrite ROM when the given settings are identical with the one already in the sensor

Parameters
a_bussoftware one wire bus context
a_romcodedevice romcode (or NULL if only one device on the bus)
