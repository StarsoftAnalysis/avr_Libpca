23 /**
24  * @file tdelay.h
25  *
26  * @brief API for timer delays
27  *
28  * This file provides a basic API for timer based delays.
29  *
30  * In order to use this API an interrupt implementation must be enabled in config.h for a requested timer (TDELAY_IMPLEMENT_T0_INT, TDELAY_IMPLEMENT_T1_INT, TDELAY_IMPLEMENT_T2_INT).
31  * For details please refer to the documentation of config.h options
32  *
33  * This API works in general the same as beeper API. The only difference between those two is that beeper API in addition toggles the OCXA pin of the associated timer in order to
34  * generate the requested frequency on the pin. tdelay API does not configure the OCXA pin as output. For all timers only TIMERX_COMPA_vect interrupt is implemented. This implementation
35  * does not take care about COMPB interrupts at all.
36  *
37  * @example tdelay_01.c
38  *
39  * General tdelay API usage example
40  *
41  * @example tdelay_02.c
42  *
43  * Using tdelay API in conjunction with beeper API
44  */
