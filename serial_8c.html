ISR ( USART_RX_vect  ,
ISR_BLOCK   
)

receive USART interrupt

Data is received in this ISR and placed in the RX ring buffer - if there is still space available. If statistics support is enabled those will be updated in this ISR as well

Parameters
USART_RX_vect

must read the data in order to clear the interrupt flag

calculate the next available ring buffer data bucket index

do not overflow the buffer

must read the data in order to clear the interrupt flag

Examples:
d7seg.c, and timer_01.c.
